Quick tips (channels, save, playback)

Choose a channel on the left — each loads a catalog JSON from final one/… (flat array or {"albums":[…]}; strict-shuffle files are not used here). Filter by album, edit fields, drag tracks to reorder. Set Artist (this track) per song, or leave it blank to use the album default. Save JSON writes the active channel file. Run python3 catalog_server.py for instant save via POST /__save_catalog__/YourFile.json. Use Add album and Add song manually to create records directly in the panel. Use ▶ on a track to play its MP3 in the bar below (requires valid mp3_url).
