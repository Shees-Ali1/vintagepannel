Quick tips (filter, save, server)

Filter by album narrows the view. Album year, thumbnail, artist, and description apply to every track in that album. Drag the handle on each song card to reorder; sequence updates when you save. For instant save into the same file on disk, run python3 catalog_server.py then open the editor URL it prints — Save JSON will update Warren_G_GFunk_Radio.json directly. Plain http.server cannot write files; then Save falls back to download / file picker.
